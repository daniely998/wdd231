WDD 231 Individual Project Site Plan

Site Name: Hong Kong Creative Writer Club

This name clearly stated the focus of the club, which is creative writers in Hong Kong.

Site Purpose

The purpose of the site is to provide information about the writer club, like works done by either club members or other noticeable work from native writers, as well as gathering information and membership application form. Possibly more.

Scenarios

What does the club do?

How do I join the club?

Can I get help publishing my own work?

How do I meet or get to know other writers?

Color Scheme

#1c3738ff for header (and footer) background, #f4fff8ff for the title, #8baaadff for the main background, #4d4847ff for the main font

Typography

Playwrite New Zealand Basic for the whole site
